1. Что такое <ol>, <ul> и <li>?
    Теги для организации списков
    ol - ordered list
    ul - unordered list
    li - list item
2. Сверстать сложную таблицу с объединением строк и столбцов.
    Нужно использовать rowspan и colspan
3. В каком регистре лучше писать HTML-код?
    В нижнем регистре. Некоторые системы регистрозависимы, некоторые нет.
4. Как убрать синюю рамку вокруг картинки-ссылки?
    Вообще я никакой синей рамки не вижу, попробовал во всех браузерах.
    В инете рекомендуют поставить в css img {border:none;}
5. Обязательно ли писать alt в <img>?
    Строго говоря, не обязательно, но является правилом хорошего тона. Выводится вместо изображения, если его не удалось загрузить.
6. Что такое html-entities?
    Специальная аббревиатура для вывода символов, зарезервированных в языке разметки. Например &gt; - это знак больше и т.д.
7. Как убрать подчеркивание для всех ссылок на странице?
    В CSS необходимо указать
        text-decoration: none;
8. Что делает свойство clear?
    Не позволяет размещать элементы слева, справа или с обеих сторон от элемента.
9. Есть ячейка таблицы, в ней 3 дива, в каждом диве – по слову. На первый див применен стиль float: left; на третий – float: right. Что нужно применить на второй див,
    чтобы все три надписи/дивы были в одной горизонтальной строке, то есть первый имел бы выравнивание по левому краю, третий – по правому, а второй – по центру?

                <div style="float: left; width: 33%;"> Лево </div>
                <div style="float: left; width: 34%; text-align: center"> Центр </div>
                <div style="float: right; width: 33%; text-align: right"> Право </div>

10. Перечислить по пять знакомых вам псевдоклассов и псевдостилей с указанием их функционала.
    hover - при наведении мыши
    active - при активировании (между нажатием и отпусканием мыши)
    visited - после посещения ссылки
    focus - элемент с фокусом
    valid - валидные input элементы

    first-child - первый элемент среди братьев
    nth-child() - выбирает элементы по правилу среди братьев
    nth-last-child() - выбирает элементы по правилу среди братьев с конца
    last-child  - последний элемент среди братьев
    only-child - элементы, которые являются единственными у родителя
