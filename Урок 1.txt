1. global - получение доступа к переменной, объявленной в глобальной области видимости.
Применимо при использовании, например, глобальных констант.

2. Суперглобальные переменные: 
$_SESSION - данные сессии
$_POST - переменные, переданные методом Post
$_GET - переменные, переданные методом Get

3. Закрывающий тег ?> используется, если после php-скриптов присутствует html

4. Что выведет программа в каждом случае и почему?
function changeX($x){
   $x += 5;
   echo $x;
}
$x = 1;
echo $x;
changeX($x);
echo $x;

Вывод: 1 6 1
Объяснение: при вызове функции внутри неё создаётся локальная копия переменной, которая при выходе из функции удаляется. Глобальная область видимости и область видимости функции не пересекаются.

5. Что выведет программа в каждом случае и почему?
function test()
{
    static $a = 0;
    echo $a;
    $a++;
}
test();
test();
test();

Вывод: 0 1 2
Объяснение: статическая переменная объявляется и инициализируется один раз и сохраняется для всех экземпляров объектов или при каждом вызове функции.
