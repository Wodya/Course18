5. Для чего нужны составные индексы в БД? Приведите примеры
    Составные индексы облегчают поиск при выборке с несколькими условиями. При этом имеет значение порядок полей в индексе.
    Например, когда ищутся сотрудники по году рождения и с определённым именем.
6. Практическая задача. Спроектируйте систему новостной ленты по указанным требованиям (нужно создать схему БД, указать необходимые индексы, перечислить стек технологий):
    Таблица news (Новости):
        id bigint primary key
        subject varchar(500)
        body varchar(max)
        create_date datetime default(GetDate)
        index(create_date)
    Таблица comment(Комментарии):
        id bigint primary key
        news_id bigint references news(id)
        user_id int not null
        create_date datetime default(GetDate)
        index(news_id, create_date)

    а.Лента новостей выводится по убыванию: от самой свежей новости к более давним;
        Делаем индекс в таблице news по полю create_date
    б.Каждая новость снабжена лентой комментариев;
        Таблица comment с внешним ключом на таблицу news
    в.Лента комментариев выводится по возрастанию: от самого давнего комментария к новым;
        Делаем индекс в таблице comment по полям news_id и create_date
    г.Обе ленты (новостей и комментариев) имеют функционал пагинации: новости – по 10 единиц, комментарии – по 25;
        Первичный ключ обеспечивает возможность пагинации
    д.Комментарии могут оставлять только авторизованные пользователи;
        Добавляем поле user_id not null
    е.Новостей и комментариев может быть очень много.
        Делаем ключи bigint
    а. Напишите запрос, который будет отвечать за пагинацию новостей или комментариев.
        n - номер страницы
        Select top(10) * from news offset n;
    б. Где лучше сортировать выводимые данные – на уровне БД или логики? Почему?
        Однозначно в БД. СУБД специально оптимизирована для работы с массивами данных с использованием индексов. При сортировке в логике невозможно использовать индексы.

10. Задано время отправления поезда и время в пути до конечной станции. Требуется написать в блокноте метод
    arrivalTime(int hours, int minutes, int hoursOnRoad, int minutesInRoad) : String

    function arrivalTime(int $hours, int $minutes, int $hoursOnRoad, int $minutesInRoad) : String{
      $resultMinute = ($minutes + $minutesInRoad) % 60;
      $totalHour = $hours + $hoursOnRoad + ($minutes + $minutesInRoad) / 60;
      $resultHour = $totalHour%24;
      $day = $totalHour > 23 ? "Завтра" : "Сегодня";
      return "Сейчас {$hours}:{$minutes}. Время прибытия {$day}: {$resultHour}:{$resultMinute}" . PHP_EOL;
    }

    echo arrivalTime(13,20,1,30);
    echo arrivalTime(19,50,5,10);
